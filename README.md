# test-frontend

This template should help get you started developing with Vue 3 in Vite.

## Recommended IDE Setup

[VSCode](https://code.visualstudio.com/) + [ES7+](https://marketplace.visualstudio.com/items?itemName=dsznajder.es7-react-js-snippets)

## Customize configuration

See [Vite Configuration Reference](https://vitejs.dev/config/).

## Features

- **Form Validation**: Uses `react-text-mask`
- **Cancel Previous Requests**: Automatically cancels previous requests when a new one is made.
- **Error Handling**: Displays error messages if no users match the search criteria or other issues occur.

## Tech Stack

- **Frontend**: React + TypeScript

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
